Block vs. Inline Elements

Resize the window to see how they behave!

1. Block Elements (The Stack)

Block elements take up the full width available. They always start on a new line.

I am a DIV (Block). I push everyone else to the next line.
I am another DIV. See how I stack?

I am a Paragraph (also Block).

2. Inline Elements (The Sentence)

Inline elements only take up as much space as they need. They sit side-by-side.

Here is some text with a SPAN (Inline) inside it. And here is a Link (Inline). They flow with the text.

3. Inline-Block (The Best of Both)

These sit side-by-side like Inline, but you can set their width and height like Block.
